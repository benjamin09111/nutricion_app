# Memoria de Correcciones y Aprendizajes (Antigravity)

Este archivo almacena exclusivamente las correcciones hechas por el usuario y los errores cometidos que no se deben repetir. Es la base de aprendizaje empírico del agente.

## Errores Cometidos (NO repetir)
- **Crear componentes redundantes:** NUNCA crear un componente o lógica nueva si ya existe una solución reutilizable en el codebase. Buscar exhaustivamente antes de codificar.
- **Campos fantasma:** No dejar que el estado gestacional persista si el sexo cambia a Masculino u Otro. Limpiar las variables del formulario inmediatamente.
- **Colores fuera de paleta:** Usar SOLO los códigos hexadecimales de la paleta definida en `globals.css`. Nunca usar colores genéricos de Tailwind (`green-500`, etc.). Los colores de marca son: Emerald `#8da84f` / `#74853f` / `#f7fbe8`, e Indigo `#8f70d8`.
